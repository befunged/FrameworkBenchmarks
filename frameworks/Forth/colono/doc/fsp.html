Forth Server Pages

Syntax

A Forth Server Pages (FSP) page is a HTML page in which Forth code has been embedded using <% ... %>.

Example: 

<html>
<head>
<title>Forth Server page</title>
<body>
<p><%
." Hello!"
%>
</html>

When an FSP page is processed, the following words are redefined so that their output appears in the generated page:

  • ."
  • TYPE
  • EMIT
  • SPACE
  • .
  • U.
  • D.

%> is implemented as a Forth word and therefore must be surrounded by whitespace.

Configuration

To enable FSP, the handler handle-fsp must be installed by the startup script: 

require ../fsp/fsp.fs

' handle-fsp S" *.fsp" add-handler

An example startup script for FSP is located at examples/fspdemo.fs (On Windows: examples/fspdemowin.fs).
